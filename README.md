# YOLO, Reimplemented

A YOLO reimplementation with sensible opinionated API and easy-to-eye code

## Installation

```bash
uv pip install git+https://github.com/ariaghora/yolo-re.git
```

## Development

```bash
git clone https://github.com/ariaghora/yolo-re.git
cd yolo-re
uv sync
uv run pytest
```
